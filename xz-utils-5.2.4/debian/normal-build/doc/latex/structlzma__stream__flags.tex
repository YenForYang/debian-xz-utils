\subsubsection{version}
{\footnotesize\ttfamily uint32\+\_\+t lzma\+\_\+stream\+\_\+flags\+::version}



Stream Flags format version. 

To prevent A\+PI and A\+BI breakages if new features are needed in Stream Header or Stream Footer, a version number is used to indicate which fields in this structure are in use. For now, version must always be zero. With non-\/zero version, the \doxyref{lzma\+\_\+stream\+\_\+header\+\_\+encode()}{p.}{stream__flags_8h_a2ebb8d6dff23daeb3de398913b845eff} and \doxyref{lzma\+\_\+stream\+\_\+footer\+\_\+encode()}{p.}{stream__flags_8h_a438249a75ea8da952a7474b92bfe7b7a} will return L\+Z\+M\+A\+\_\+\+O\+P\+T\+I\+O\+N\+S\+\_\+\+E\+R\+R\+OR.

\doxyref{lzma\+\_\+stream\+\_\+header\+\_\+decode()}{p.}{stream__flags_8h_ae03198e464f0d296e601ff841e100805} and \doxyref{lzma\+\_\+stream\+\_\+footer\+\_\+decode()}{p.}{stream__flags_8h_aa92a383f85753bb79ee23227fa68186c} will always set this to the lowest value that supports all the features indicated by the Stream Flags field. The application must check that the version number set by the decoding functions is supported by the application. Otherwise it is possible that the application will decode the Stream incorrectly. \mbox{\label{structlzma__stream__flags_aaa65ed7a55a098f829f04dba25d0f212}}

\subsubsection{backward\+\_\+size}
{\footnotesize\ttfamily \textbf{ lzma\+\_\+vli} lzma\+\_\+stream\+\_\+flags\+::backward\+\_\+size}



Backward Size. 

Backward Size must be a multiple of four bytes. In this Stream format version, Backward Size is the size of the Index field.

Backward Size isn\textquotesingle{}t actually part of the Stream Flags field, but it is convenient to include in this structure anyway. Backward Size is present only in the Stream Footer. There is no need to initialize backward\+\_\+size when encoding Stream Header.

\doxyref{lzma\+\_\+stream\+\_\+header\+\_\+decode()}{p.}{stream__flags_8h_ae03198e464f0d296e601ff841e100805} always sets backward\+\_\+size to L\+Z\+M\+A\+\_\+\+V\+L\+I\+\_\+\+U\+N\+K\+N\+O\+WN so that it is convenient to use \doxyref{lzma\+\_\+stream\+\_\+flags\+\_\+compare()}{p.}{stream__flags_8h_a3e25ca4205021302882a696283d45263} when both Stream Header and Stream Footer have been decoded. \mbox{\label{structlzma__stream__flags_ab1052ea7047c8d67f127f33278166647}}
