\subsection{Detailed Description}
Filter options. 

This structure is used to pass Filter ID and a pointer filter\textquotesingle{}s options to liblzma. A few functions work with a single \doxyref{lzma\+\_\+filter}{p.}{structlzma__filter} structure, while most functions expect a filter chain.

A filter chain is indicated with an array of \doxyref{lzma\+\_\+filter}{p.}{structlzma__filter} structures. The array is terminated with .id = L\+Z\+M\+A\+\_\+\+V\+L\+I\+\_\+\+U\+N\+K\+N\+O\+WN. Thus, the filter array must have L\+Z\+M\+A\+\_\+\+F\+I\+L\+T\+E\+R\+S\+\_\+\+M\+AX + 1 elements (that is, five) to be able to hold any arbitrary filter chain. This is important when using \doxyref{lzma\+\_\+block\+\_\+header\+\_\+decode()}{p.}{block_8h_a7f5487c21a7b36a8bd17be36074d43c9} from \doxyref{block.\+h}{p.}{block_8h}, because too small array would make liblzma write past the end of the filters array.
